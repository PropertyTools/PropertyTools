-------------------------------------------

2014.1.{build.counter}.1

Artifact paths:

output/**/*.zip => Output.zip
packages/**/*.* => Packages.zip

https://hg.codeplex.com/propertytools

Build steps

Update version number
Tools\Lynx
VersionUpdater.exe
/Version=%build.number% /Directory=..\..

echo %build.vcs.number%> Output\changeset.id

Build .NET 4.0 solution
Source\PropertyTools_NET40.sln

Build .NET 4.5 solution
SourcePropertyTools_NET45.sln

Run unit tests
NUnit runner: 
NUnit 2.6.3

Version v4.0
Run tests from:
Source\PropertyTools.Wpf.Tests\bin\Release\NET40\*.Tests.dll
.NET Coverage tool: JetBrains dotCover
Filters:
+:PropertyTools*
-:*.Tests*

Pack NuGet
build
pack.cmd

Delete pdb files
Output\NET45\Examples
del /S *.pdb

Create release
Build
ZipRelease.cmd %build.number%
